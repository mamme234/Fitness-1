# Pro Gym App

Welcome to **Pro Gym App** – a comprehensive full-stack MERN application designed to track workouts, monitor nutrition, manage progress, and challenge users to reach their fitness peaks.

## Features
* **User Authentication:** Secure JWT-based registration and login.
* **Workout Tracker:** Create, read, update, and delete custom workout routines.
* **Nutrition Logs:** Track daily caloric intake and macro distributions.
* **Progress Dashboard:** Monitor weight changes and fitness achievements over time.
* **Challenges:** Join community fitness challenges to stay motivated.
